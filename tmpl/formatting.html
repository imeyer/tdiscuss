Formatting Guide

Thread Subjects

Subjects are plain text only. All HTML and markdown is stripped for clean, consistent titles.

You type: Check out this **cool** thing!
Result:
{{ .SubjectExample }}

Post Bodies

Bodies support some markdown. Headings (h1-h6) are not allowed to keep discussions readable.

Text Formatting
**bold** → bold
*italic* → italic
~~strikethrough~~ → strikethrough
`inline code` → inline code
Links & Images
[link text](https://example.com) → link text
![alt text](https://example.com/img.png) → (image displayed)
Lists
You type: 
- Item one
- Item two
  - Nested item
Result:
  • Item one
  • Item two
    • Nested item
You type: 
1. First
2. Second
3. Third
Result:
  1. First
  2. Second
  3. Third
Code Blocks
You type: 
```go
func main() {
    fmt.Println("Hello!")
}
```
Result: 
func main() {
    fmt.Println("Hello!")
}
Blockquotes
You type: 
> This is a quote
> from someone wise
Result:

This is a quote
from someone wise

Nested quotes (for reply chains): 
> Alice wrote:
> > Bob wrote:
> > > Original message
> > Bob's reply
> Alice's reply
My response
Result:

Alice wrote:

Bob wrote:

Original message

Bob's reply

Alice's reply

My response

Tip: More > symbols = deeper nesting. Keep the same prefix on all lines that belong together. Blank lines between quoted lines create separate blockquotes.

What's NOT Allowed

Headings and tables are stripped to keep discussions readable:

# Big Heading → Big Heading (stripped)
## Medium Heading → Medium Heading (stripped)
| table | syntax | → (stripped)
